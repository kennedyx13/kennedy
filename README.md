Compiling Kennedy: QT4 Dependencies
--------------------

**Daemon | Kennedyd**





    $ git clone https://yalta1945@bitbucket.org/yalta1945/john.git 
    $ sudo apt-get update
    $ cd john/src
    $ make -f makefile.unix USE_UPNP=1
    $ strip Kennedyd
    $ ./Kennedyd
